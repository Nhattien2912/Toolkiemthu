# Web Test Catalog

Tai lieu nay la bo test catalog noi bo duoc chuyen hoa va paraphrase tu checklist tham khao:
[Software Testing Help - 180+ Test Cases Example for Web Application Testing](https://www.softwaretestinghelp.com/sample-test-cases-testing-web-desktop-applications/)

Muc tieu:
- Dua bo checklist tong hop vao project de de lap ke hoach test.
- Tach ro giua test co the tu dong hoa va test nen manual.
- Lam co so de tiep tuc code Selenium/pytest cho app hien tai.

Ghi chu:
- Day la ban chuyen hoa noi bo, khong phai ban sao chep nguyen van.
- Khong phai test nao cung nen dua vao Selenium.
- Cac test duoi day duoc viet theo huong ap dung cho web app tong quat, sau do map dan vao Guangli Shop.
